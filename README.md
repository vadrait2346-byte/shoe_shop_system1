# Информационная система "Учет обуви на складах"

## Описание проекта
Данная информационная система разработана в рамках выполнения требований к ГИА ДЭ по специальности "Информационные системы и программирование". Приложение предназначено для автоматизации контроля остатков товаров на обувных складах и управления заказами.

## Реализованный функционал
- **Разграничение прав доступа**: Гость, Клиент, Менеджер, Администратор.
- **База данных 3НФ**: Реализована на СУБД SQLite, содержит 7 реляционных таблиц, связанных внешними ключами.
- **Модуль миграции данных**: Автоматический парсинг, нормализация и очистка плоских экзаменационных таблиц Excel заказчика (`Tovar.xlsx`, `user_import.xlsx`, `Заказ_import.xlsx`).
- **Интерфейс пользователя (PyQt6)**: Динамическая многокритериальная фильтрация каталога обуви (по брендам и размерам) и полный цикл CRUD-операций для администратора.

## Технологический стек
- **Язык**: Python 3.10+
- **Интерфейс**: PyQt6
- **База данных**: SQLite3
- **Инструменты**: openpyxl, DBeaver (визуализация ER-диаграммы)

## Инструкция по установке и запуску

1. Клонируйте репозиторий проекта.
2. Создайте и активируйте виртуальное окружение:
   ```bash
   python -m venv venv
   venv\Scripts\activate
   ```
3. Установите необходимые зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Выполните первичный импорт экзаменационных файлов Excel:
   ```bash
   python import_excel.py
   ```
5. Запустите основное приложение:
   ```bash
   python main.py
   ```
